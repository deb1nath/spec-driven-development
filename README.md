## Spec Driven Development plugin

To install the plugin in your working project execute the following command to add this plugin in your claude code
- ```/plugin marketplace add https://github.com/deb1nath/spec-driven-development.git```

- ```/plugin install agentic-sdlc@spec-driven-development```

# Then in Claude Code:
Add to marketplace then run the command

- ```@supervisor implement the feature spec at docs/specs/features/FEAT-001.md```
